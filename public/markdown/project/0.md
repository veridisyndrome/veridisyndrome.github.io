**Overview:** FlightRadar is a Java-based flight tracking application developed as part of my CS108 semester project at EPFL. The application features a JavaFX user interface and implements signal processing algorithms to visualize real-time flight data.

- **Key Features**:
  - **JavaFX UI**: Interactive and user-friendly interface for real-time flight visualization.
  - **Signal Processing**: Advanced algorithms to process and display flight data accurately.
  - **Map Caching**: Efficient caching mechanism for map tiles, enhancing performance.

- **Project Management**:
  - **Version Control**: Leveraged Git for version control, ensuring smooth collaboration and a clear project history.
  - **Task Management with Notion**: Tasks and milestones were organized in Notion, allowing for clear tracking of progress and deadlines, and ensuring alignment among team members.

  - **Algorithm Development**: Developed and optimized signal processing algorithms for real-time applications.
  - **Project Collaboration**: Strengthened teamwork and communication skills through collaborative project development.
