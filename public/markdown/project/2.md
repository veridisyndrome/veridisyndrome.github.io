**Overview:** ImgFS is an image-based file system inspired by Facebook's Haystack architecture, implemented as part of a collaborative C programming project. This system-level application efficiently manages images within a custom file system, providing optimized storage and retrieval solutions for large-scale image handling.

- **Key Features**:
  - **Haystack-Inspired Architecture**: Designed to address the challenges of managing millions of images, with features like image deduplication, multiple resolution management, and reduced file handling.
  - **Efficient Image Management**: Stores multiple images in a single file, combining data and metadata for rapid access.
  - **Real-Time Performance**: Utilizes multithreading to handle concurrent operations, improving system performance and responsiveness.

- **Project Management**:
  - **Agile-Inspired Approach**: The team managed the project using an Agile-like methodology with clearly defined phases and goals.
  - **Version Control**: The team employed Git for tracking changes and managing code versions throughout the project.
  - **Task Management with Notion**: Tasks and deadlines were organized and tracked collaboratively using Notion.

- **Skills Developed**:
  - **C Programming**: Focused on low-level system programming in C.
  - **Image Processing**: Implemented efficient methods for storing and retrieving images.
  - **Collaborative Project Management**: The team managed all aspects of the project, from concept to implementation.
