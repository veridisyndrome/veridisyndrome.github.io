**Overview:** Scala-Doc is an online word processor inspired by Google Docs, developed as part of my CS214 course at EPFL. This solo project was implemented in Scala, focusing on real-time collaborative editing, while the HTML/CSS for the interface was provided.

- **Key Features**:
  - **Real-Time Collaboration**: Enables multiple users to edit documents simultaneously with instant updates.
  - **Rich Text Editing**: Supports various text formatting options.
  - **Cross-Platform**: Functional across different devices and operating systems.

- **Project Management**:
  - **Agile-Inspired Approach**: Self-managed the project using an Agile-like method with set milestones and progress checks.
  - **Version Control**: Used Git for version tracking.
  - **Task Management with Notion**: Organized and tracked tasks and deadlines.

- **Skills Developed**:
  - **Scala Programming**: Focused on Scala and functional programming principles.
  - **Collaborative Editing**: Developed real-time data handling for multiple users.
  - **Independent Project Management**: Managed all aspects of the project independently.

- **Technologies Used**:
  - **Scala**
  - **Git**
  - **Notion**
